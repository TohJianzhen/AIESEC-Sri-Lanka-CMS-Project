## Installation

Quickly describe how to install your project and how to get it running

1. Install Node dependencies

		npm install

	Express is a Node.js web application framework
		npm i express

	Create a session middleware with the given options
		npm i express-session

	Node.js body parsing middleware.
		npm i body-parser

	Prints the augmented PATH to the console 
		npm i path

	save automatically writes to your package.json file
		npm i ejs-mate

	A node.js driver for mysql.
		npm i mysql

	Dotenv is a zero-dependency module that loads environment variables from a . env file into process. env
		npm i dotenv

	JSON Web Token (JWT) for securely transmitting information between parties as a JSON object
		npm i jsonwebtoken

	A library to help you hash passwords.
		npm i bcryptjs


2. Create Datababse

        First create customers database
	Then import customers.sql 

3. run Node project

        node app.js
